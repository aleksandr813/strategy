# Тест-кейс «Функциональность "Забыли пароль?"» - №02_04

| **ID** | **Описание** | **Приоритет** | **Предусловия** | **Шаги воспроизведения** | **Ожидаемый результат** |
| :-: | :-: | :-: | :-: | :-: | :-: |
| **04_01** | Проверка перехода по ссылке "Забыли пароль?" | Средний | Пользователь на экране входа | 1. Нажать на ссылку "Забыли пароль?"<br>2. Дождаться реакции системы | 1. Открывается окно/страница восстановления пароля<br>2. Отображается поле для ввода логина<br>3. Есть кнопка "Восстановить" |
