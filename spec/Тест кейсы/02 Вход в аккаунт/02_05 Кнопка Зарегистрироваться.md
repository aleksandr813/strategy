# Тест-кейс «Кнопка Зарегистрироваться» - №02_05

| **ID** | **Описание** | **Приоритет** | **Предусловия** | **Шаги воспроизведения** | **Ожидаемый результат** |
| :-: | :-: | :-: | :-: | :-: | :-: |
| **05_01** | Проверка перехода на регистрацию | Высокий | Пользователь на экране входа | 1. Нажать на кнопку "Создать аккаунт"<br>2. Дождаться перехода | 1. Происходит переход на экран регистрации<br>2. Отображаются все поля регистрации<br>3. URL соответствует странице регистрации |
