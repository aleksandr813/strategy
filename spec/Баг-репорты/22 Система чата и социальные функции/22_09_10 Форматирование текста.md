# Баг-репорт - Форматирование текста

**Статус:** Открыт  
**Приоритет:** Высокий  
**Дата начала:** 07.12.2025  
**Дата окончания:**    
**Environment:** Google Chrome (последняя версия)   

## Описание

При вводе и отправке длинных сообщений в чате (от ~300 символов) панель чата растягивается на весь экран, блокируя видимость других элементов интерфейса. Отсутствует ограничение на количество вводимых символов, а при отправке очень длинных сообщений (~20000 символов) возникает ошибка 9000.

## Предусловия

1. Открыт чат

## Шаги воспроизведения

1. Ввести сообщение длиной примерно 300 символов (без пробелов или с минимальными пробелами)
2. Отправить сообщение
3. Наблюдать за поведением интерфейса чата
4. Повторить с сообщением длиной ~20000 символов

## Фактический результат

- Панель чата растягивается на весь экран
- Сообщение отображается в одну строку без переносов
- Другие элементы интерфейса становятся невидимыми или перекрываются
- При отправке очень длинного сообщения возникает ошибка 9000
- В приветственном сообщении вместо ника игрока отображается "12321312eqweqw"

## Ожидаемый результат

- Длинные сообщения должны автоматически переноситься на новые строки
- Панель чата должна сохранять фиксированные размеры
- Должно быть ограничение на длину сообщений (например, 1000 символов)
- При превышении лимита должно появляться предупреждение, а не ошибка 9000
- В приветствии должен использоваться корректный никнейм игрока
