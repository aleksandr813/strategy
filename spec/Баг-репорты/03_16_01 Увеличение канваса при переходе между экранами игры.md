# Баг-репорт - Самопроизвольное увеличение канваса при переходе между экранами игры

**Статус:** Провален  
**Приоритет:** Высокий  
**Серьезность:** Средняя  
**Дата начала:** 06.11.2025  
**Дата окончания:**   
**Environment:** Google Chrome (последняя версия) 

## Описание

При переходе на другие экраны игры ("Битва", "Чат", "Глобальная карта" и т.д.) и последующем возврате в менеджмент деревни канвас непроизвольно увеличивается на весь экран. Проблема не воспроизводится при переключении вкладок браузера.

## Предусловия

- Открыт менеджмент деревни

## Шаги воспроизведения

1. Находиться в менеджменте деревни с корректно отображаемым канвасом
2. Нажать на любую кнопку перехода к другому экрану игры ("Битва", "Чат", "Глобальная карта")
3. Дождаться загрузки выбранного экрана
4. Вернуться в менеджмент деревни
5. Наблюдать за размером канваса

## Фактический результат

- Канвас увеличивается на весь экран после возврата в менеджмент деревни
- Нарушается интерфейса
- Требуется полное обновление страницы для восстановления нормального размера

## Ожидаемый результат

- При возврате в менеджмент деревни канвас должен сохранять исходный размер
- Интерфейс должен отображаться в корректном состоянии без необходимости обновления страницы
- Размер канваса не должен изменяться при переходах между экранами игры
