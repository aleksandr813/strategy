# Баг-репорт - Конфликт зон размещения при переключении

**Статус:** Выполнен  
**Приоритет:** Высокий   
**Дата начала:** 15.11.2025  
**Дата окончания:** 19.11.2025  
**Environment:** Google Chrome (последняя версия) 

## Описание

При переключении между размещением юнитов и зданий в менеджменте деревни, зона размещения предыдущего объекта не скрывается. Если выбрать юнита для размещения (появляется зеленая зона), а затем переключиться на выбор здания, зона юнитов остается на экране.

## Предусловия

- Открыт менеджмент деревни
- Достаточно монет для покупки юнитов/зданий

## Шаги воспроизведения

1. В панели управления юнитами выбрать любого юнита для размещения
2. Навести курсор на игровое поле - появится зеленая зона размещения юнита
3. Не размещая юнита, перейти в панель строительства
4. Выбрать любое здание для размещения
5. Наблюдать за поведением зон размещения

## Фактический результат

- Зона размещения юнита остается видимой на экране
- Одновременно отображается зона размещения здания
- Зона юнитов исчезает только после успешного размещения здания

## Ожидаемый результат

- При выборе нового объекта для размещения (юнит/здание) предыдущая зона размещения должна немедленно скрываться
- На экране должна быть активна только одна зона размещения в любой момент времени
- Переключение между режимами размещения должно быть корректным и без визуальных конфликтов

## Результат ре-тестирования (20.11.2025)

Баг исправлен. Разные зоны выделения при переключении с зоны строительства на зону юнита и обратно пропадают.

## Комментарий тестировщика

Функциональность работает согласно ожиданиям. Повторное тестирование пройдено успешно.
