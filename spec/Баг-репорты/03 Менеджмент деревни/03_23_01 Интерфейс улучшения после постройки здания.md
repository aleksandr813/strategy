# Баг-репорт - Интерфейс улучшения после постройки здания

**Статус:** Открыт  
**Приоритет:** Высокий  
**Дата начала:** 17.12.2025  
**Дата окончания:**    
**Environment:** Google Chrome (последняя версия)  

## Описание

После покупки и размещения стены на игровом поле интерфейс автоматически открывает меню улучшения этой стены.

## Предусловия

1. Открыт менеджмент деревни
2. Достаточно монет для покупки стены

## Шаги воспроизведения

1. Открыть меню покупки зданий
2. Выбрать стену для покупки
3. Разместить стену на доступной клетке игрового поля
4. Наблюдать за поведением интерфейса после размещения

## Фактический результат

- Сразу после размещения стены автоматически открывается меню улучшения этой стены
- Необходимо закрывать меню улучшения для возврата

## Ожидаемый результат

- После размещения стены интерфейс должен оставаться в менеджменте деревни
- Меню улучшения не должно открываться автоматически
