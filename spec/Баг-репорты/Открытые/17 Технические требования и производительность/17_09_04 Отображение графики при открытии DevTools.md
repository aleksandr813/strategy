# Баг-репорт - Отображение графики при открытии DevTools

**Статус:** Открыт  
**Приоритет:** Высокий  
**Дата начала:** 17.12.2025  
**Дата окончания:**    
**Environment:** Google Chrome (последняя версия)  

## Описание

При открытии инструментов разработчика через F12 или контекстное меню «Посмотреть код» происходит деформация спрайтов зданий. Спрайты стен и других зданий искажаются.

## Предусловия

1. Открыт менеджмент деревни
2. Размещены здания - Стена, Ратуша, Башня, Шахта, Казарма

## Шаги воспроизведения

1. Открыть менеджмент деревни
2. Открыть инструменты разработчика одним из способов:
   - Нажать клавишу F12
   - Нажать ПКМ → «Посмотреть код»
4. Наблюдать за спрайтами зданий

## Фактический результат

- Спрайты зданий деформируются
- Визуальное отображение искажается

## Ожидаемый результат

- Открытие инструментов разработчика не должно влиять на отображение игровой графики
- Спрайты зданий должны оставаться четкими и неизменными
- Интерфейс не должен деформироваться при любых действиях пользователя
- Графика должна корректно масштабироваться и отображаться независимо от состояния DevTools
