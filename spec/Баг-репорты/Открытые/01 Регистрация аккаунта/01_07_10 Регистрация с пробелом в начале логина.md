# Баг-репорт - Регистрация с пробелом в начале логина

**Статус:** Открыт  
**Приоритет:** Высокий  
**Дата начала:** 17.12.2025  
**Срок окончания:**  
**Environment:** Google Chrome (последняя версия)  

## Описание

При попытке регистрации с неверными данными - система возволяет зарегистрировать пользователя.

## Предусловия



## Шаги воспроизведения

1. В поле "Логин" ввести некорректные данные (например: ` User123`)
2. В поле "Пароль" ввести любой корректный пароль
3. В поле "Подтверждение пароля" ввести пароль из пункта "Пароль"
4. В поле "Имя" ввести любое корректное имя
5. Нажать кнопку "Зарегистрироваться"
6. Наблюдать за ошибкой

## Фактический результат

Система повзоялет зарегистрировать пользователя и никак не припятствует этому.

## Ожидаемый результат

Система должна возвращать ошибку валидации:
- Логин не может содердать пробелы
