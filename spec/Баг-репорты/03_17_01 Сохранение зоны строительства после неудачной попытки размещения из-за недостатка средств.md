# Баг-репорт - Сохранение зоны строительства после неудачной попытки размещения из-за недостатка средств

**Статус:** Открыт  
**Приоритет:** Высокий   
**Дата начала:** 03.11.2025   
**Дата окончания:**    
**Environment:** Google Chrome (последняя версия)  

## Описание

При попытке строительства здания с недостаточным количеством монет зеленая зона выделения (показывающая доступные для размещения клетки) не исчезает после неудачной попытки размещения.

## Предусловия

- Открыт менеджмент деревни
- Баланс монет недостаточен для строительства выбранного здания
- Выбрано здание для строительства в панели строительства

## Шаги воспроизведения

1. Выбрать любое здание для строительства из панели строительства
2. Убедиться, что отображается зеленая зона доступного размещения
3. Кликнуть по любой доступной клетке в зеленой зоне для размещения здания
4. Наблюдать за поведением зеленой зоны выделения

## Фактический результат

- Зеленая зона выделения остается активной после неудачной попытки строительства
- Пользователь может продолжать кликать по клеткам, но здание не размещается
- Отсутствует четкая визуальная обратная связь о невозможности строительства
- Интерфейс выглядит так, будто строительство возможно

## Ожидаемый результат

- При неудачной попытке строительства из-за недостатка средств зеленая зона выделения должна немедленно исчезать
- Должно отображаться сообщение об ошибке "Недостаточно средств"
- Интерфейс должен переходить в нейтральное состояние до появления достаточных средств
- Пользователь должен получить четкую визуальную обратную связь о невозможности операции
